# Knowledge

Task-specific lessons learned from real struggles. This is the most
frequently-written part of the knowledge base.

Each file captures a non-obvious solution that someone will hit again —
the kind of thing where you tried 2+ approaches before finding one that
worked, or the root cause was surprising.

## What Belongs Here

- Lessons where the fix was non-obvious
- Problems that crossed tool/library boundaries with poor docs
- Root causes that were surprising (not just typos or missing imports)

## What Does NOT Belong Here

- Fixes obvious from the error message
- One-off configurations unique to a single project
- Problems well-covered by official docs (link instead)
- "Read the docs" with extra steps
